Release Notes

v0.4.3

  • AI Markup Diagnostics: Added clearer cross-platform diagnostics for AI Markup request, credential, model, endpoint, and connectivity failures.
  • Sanitized Troubleshooting: Added sanitized diagnostic capture and export for support sharing.
  • Support Menu: Added User Manual and About to the gear menu, with installed-version and release-note visibility.

v0.4.2

  • Error Display: Fixed error messages to show actual API error text instead of a generic invalid-response message.

v0.4.1

  • API Endpoint Update: Updated CBRD API to the production endpoint.
  • Error Handling: Improved error message parsing for responses that provide a msg field.

v0.4.0

  • Ref-to-Link Action: Convert selected <ref> elements to CBRD links and replace existing <ptr> safely.
  • CBRD Preferences: Configurable endpoint, referer, and timeout settings.
  • Validation and Errors: Clear feedback for invalid selections, missing components, and API failures.

v0.3.0

  • Complete Java Migration: Migrated the implementation from JavaScript to pure Java for better performance and reliability.
  • Enhanced UTF-8 Tools: Improved file-encoding validation and conversion with automatic backup creation.

v0.2.3

  • Oxygen integrated options/preferences page for setting the fine-tuned LLM models and API key.

v0.2.2

  • Options saved with Oxygen's wsOptionsStorage approach.
  • API key saved and retrieved with better security by Oxygen's secretOption.

v0.2.1

  • Fixed an API-key handling issue that caused an error after saving options.

v0.2.0

  • Added multi-language support for the plugin.
  • Added a custom options/preferences page for LLM models and API key configuration.

v0.1.0

  • Initial AI-driven markup for unmarked references.
  • Initial TEI tag-removal support.
