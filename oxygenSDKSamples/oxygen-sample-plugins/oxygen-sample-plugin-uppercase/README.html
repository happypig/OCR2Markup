This plugin converts every letter in the selected text to uppercase. This works only for letters.

The plugin uses the following API classes:

ro.sync.exml.plugin.selection.SelectionPluginContext;
ro.sync.exml.plugin.selection.SelectionPluginExtension;
ro.sync.exml.plugin.selection.SelectionPluginResult;
ro.sync.exml.plugin.selection.SelectionPluginResultImpl;

After installing the plugin it, can be accessed through the editor's contextual menu by choosing Plugins->UpperCase.

Example:

Input:
	tHis is a SamPle.
Result:
	THIS IS A SAMPLE.
