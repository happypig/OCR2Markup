Sample plugins for oXygen standalone distribution

The contents of this project

There are several categories of oXygen plugins:

Plugins that process the current text selection from the Text mode:

  • oxygen-sample-plugin-capitalize-lines
  • oxygen-sample-plugin-conversion
  • oxygen-sample-plugin-format-preserve-text
  • oxygen-sample-plugin-formsentences
  • oxygen-sample-plugin-formwords
  • oxygen-sample-plugin-lowercase
  • oxygen-sample-plugin-uppercase
  • oxygen-sample-plugin-zeroindent

Plugins that access the oXygen IDE (internal file editors, pages, Author mode document object model, toolbars, menu items, etc..):

  • oxygen-sample-plugin-workspace-access
  • oxygen-plugin-startup

Plugin that accesses the oXygen options:

  • oxygen-sample-plugin-impose-options

Plugin that suply an external transfomer for XSLT of XQuery:

  • oxygen-sample-plugin-impose-options

Build and deployment

Note:To build the entire set of plugins into a single archive use the project: bundle-plugins. You need to run the following command in the main project's directory:

 mvn -pl bundle-plugins -am install

The .zip file is built in the target/ directory.

To deploy it, you can unarchive the file in an oXygen installation plugins folder by running the following command in the main project's directory:

  mvn -pl bundle-plugins -am deploy

Automatic deployment into an Oxygen instance

The following command will both pack the plugin and copy it inside the plugins folder of an an oXygen instance:

mvn install -Doxygen.plugins.dir="D:\OxygenInstallDir\plugins"

This command is particular useful when you need to repeatably make changes in a plugin and afterwards deploy it and test it in an Oxygen instance.
