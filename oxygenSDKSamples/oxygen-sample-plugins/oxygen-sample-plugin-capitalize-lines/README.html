This plugin capitalizes the first letter found on every new line that is selected. Only the first letter is affected, the rest of the line remains the same. If the first character on the new line is not a letter then no changes are made.

The plugin uses the following API classes:
ro.sync.exml.plugin.selection.SelectionPluginContext;
ro.sync.exml.plugin.selection.SelectionPluginExtension;
ro.sync.exml.plugin.selection.SelectionPluginResult;
ro.sync.exml.plugin.selection.SelectionPluginResultImpl;

After installing the plugin, it can be accessed through the editor's text page contextual menu by choosing Plugins->Capitalize Lines.

Example:

Input:
	this 
	is 
	a sample.
Result:
	This 
	Is 
	A sample.
