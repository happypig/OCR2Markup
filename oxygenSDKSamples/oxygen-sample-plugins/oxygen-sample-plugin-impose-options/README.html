This plugin imposes options and layouts to oXygen.

Once installed, the plugin:

  1. adds the file templates from the "templates" directory to the oXygen "New templates" templates.
  2. applies the options from the "options.xml" file to oXygen options, at the first start after plugin is installed.
  3. applies the layout from the "application.layout" file to oXygen as a fixed layout, at each application start.

This plugin uses the following API:

ro.sync.exml.options.PerspectivesLayoutInfo;
ro.sync.exml.plugin.workspace.WorkspaceAccessPluginExtension;
ro.sync.exml.workspace.api.standalone.StandalonePluginWorkspace;

Note: To configure the templates, make a templates directory in the plugin's main directory and copy here the files you intend to use as templates.

Note: To create an options.xml file, use the action Export Global Options.. from the oXygen Options menu. Export this file to the plugin's main directory and edit it as needed.

Note: To create an application.layout file, use the action Save Layout.. from the oXygen Window menu. Save this file into the plugin's main directory and edit it as needed.
