This plugin converts every character in the selected text to lowercase. Of course, this works only for letters.

The plugin uses the following API classes:

ro.sync.exml.plugin.selection.SelectionPluginContext;
ro.sync.exml.plugin.selection.SelectionPluginExtension;
ro.sync.exml.plugin.selection.SelectionPluginResult;
ro.sync.exml.plugin.selection.SelectionPluginResultImpl;

After installing the plugin, it can be accessed through the editor's contextual menu by choosing Plugins->LowerCase.

Example:

Input:
	THIS is A SaMple.
Result:
	this is a sample.
