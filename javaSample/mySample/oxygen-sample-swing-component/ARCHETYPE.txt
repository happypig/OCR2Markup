How to convert the current project to a Maven Archetype
=======================================================

http://www.luckyryan.com/2013/02/15/create-maven-archetype-from-existing-project/

First, in the project directory, run the command:

{{{
mvn archetype:create-from-project
}}}

You should get a SUCCESS here.
Next, copy the '''target/generated-sources/archetype''' to a new directory.
The new directory structure is:

{{{
- pom.xml
- src/
- target/
}}}

Use 
{{{ 
 mvn clean
}}} 

to remove the target directory.
