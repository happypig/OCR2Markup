This plugin installs a redirect handler for an URL opened in <oXygen. Such a plugin must provide an implementation of ro.sync.exml.plugin.openredirect.OpenRedirectExtension which can redirect opened URLs in Oxygen.

How the plugin works:

  1. You can decide that when a ZIP archive gets opened a file from the archive must be automatically opened as well.
  2. When a SVG gets opened it is opened both in the main editor and in the image previewer.
  3. Open a totally different location from the original selected URL.

For more information: http://www.oxygenxml.com/doc/ug-oxygen/index.html?q=/doc/ug-oxygen/concepts/open-redirect-plugin.html
